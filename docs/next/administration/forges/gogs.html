Gogs

Configuration‚Äã

This is a full list of configuration options. Please note that many of these options use default configuration values that should work for the majority of installations.

WOODPECKER_GOGS‚Äã

Default: false

Enables the Gogs driver.

WOODPECKER_GOGS_URL‚Äã

Default: https://try.gogs.io

Configures the Gogs server address.

WOODPECKER_GOGS_GIT_USERNAME‚Äã

Default: empty

This username is used to authenticate and clone all private repositories.

WOODPECKER_GOGS_GIT_USERNAME_FILE‚Äã

Default: empty

Read the value for WOODPECKER_GOGS_GIT_USERNAME from the specified filepath

WOODPECKER_GOGS_GIT_PASSWORD‚Äã

Default: empty

The password is used to authenticate and clone all private repositories.

WOODPECKER_GOGS_GIT_PASSWORD_FILE‚Äã

Default: empty

Read the value for WOODPECKER_GOGS_GIT_PASSWORD from the specified filepath
